## APA formatting rules (quick)

| Element | APA 7 rule |
|---------|------------|
| **Reference list title** | Centered heading: **References** (not “Bibliography”) |
| **Order** | Alphabetical by first author’s surname |
| **Indent** | Hanging indent: first line flush left; following lines indented 0.5 in |
| **In-text** | Author–date: (Breiman, 2001) or Breiman (2001) |
| **3+ authors** | First citation and all later: (Tavallaee et al., 2009) |
| **Software** | `Author. (Year). Name of software (Version) [Computer software]. URL` |
| **Same author, same year** | Add letters: (OpenJS Foundation, 2024a), (OpenJS Foundation, 2024b) |
| **DOI/URL** | Include when available; no “Retrieved from” unless needed |

**Example paragraph (report):**

> NetGuard AI classifies network flows using a Random Forest classifier (Breiman, 2001) implemented with scikit-learn (Pedregosa et al., 2011). The model was trained on the NSL-KDD dataset (Tavallaee et al., 2009), which refines the earlier KDD Cup benchmark (Lippmann et al., 2000). Live traffic is aggregated with Scapy (Biondi, n.d.) and sent to a FastAPI inference service (Ramírez, n.d.) via an Express backend (OpenJS Foundation, n.d.-a) before results are stored in PostgreSQL (PostgreSQL Global Development Group, n.d.) and displayed in a React dashboard (Meta Platforms, Inc., n.d.).
